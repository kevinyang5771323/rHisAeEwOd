# 家政服务管理平台系统介绍 java102

## 项目概述

家政服务管理平台是一个基于前后端分离架构的 web 网站，主要服务于管理员、用户和服务人员三个角色。通过此平台，用户可以便捷地获取家政服务信息，管理员可以对服务流程进行高效管理，服务人员则可及时更新服务状态。

## 技术栈

- 后端：Java、SSM 框架、Spring Boot
- 前端：Vue.js
- 数据库：MySQL
- 服务器：Apache Tomcat
- 项目管理工具：Maven
- 开发工具：IntelliJ IDEA

## 功能模块

### 用户功能模块

- 首页浏览
- 服务信息查询
- 公告信息查看
- 留言反馈
- 个人中心管理

### 管理员功能模块

- 首页总览
- 个人中心管理
- 用户管理
- 服务人员管理
- 服务信息管理
- 服务类型管理
- 服务预约管理
- 服务取消管理
- 服务分配管理
- 服务进度管理
- 评价信息管理
- 留言反馈处理
- 系统管理

### 服务人员功能模块

- 首页浏览
- 个人中心管理
- 服务分配查看
- 服务进度更新
- 评价信息查看

## 系统角色

- 管理员：负责整个平台的服务流程管理、用户和服务人员管理等。
- 用户：可查询服务信息、预约服务、发表评价等。
- 服务人员：可查看分配的服务、更新服务进度、回复评价等。

## 运行环境

- 后端：JDK 1.8 或以上版本，MySQL 5.7 或以上版本，Apache Tomcat 8.5 或以上版本
- 前端：现代浏览器（建议使用 Chrome、Firefox 或 Safari）
